2. Calendar (with FullCalendar Integration)

Functionality:
	•	Display all bookings from the bookings table (inside urban_council DB).
	•	Enable Add/Edit/Delete via modal or inline form.

Steps:
	1.	Install FullCalendar (via CDN or npm):

<link href='https://cdn.jsdelivr.net/npm/fullcalendar@6.1.8/main.min.css' rel='stylesheet' />
<script src='https://cdn.jsdelivr.net/npm/fullcalendar@6.1.8/main.min.js'></script>


	2.	Fetch Booking Events from PHP/MySQL:

$query = "SELECT * FROM bookings";
$result = mysqli_query($conn, $query);
$events = [];
while($row = mysqli_fetch_assoc($result)) {
    $events[] = [
        'id' => $row['id'],
        'title' => $row['title'],
        'start' => $row['start_time'],
        'end' => $row['end_time']
    ];
}
echo json_encode($events);


	3.	Render Calendar:

document.addEventListener('DOMContentLoaded', function() {
    var calendar = new FullCalendar.Calendar(document.getElementById('calendar'), {
        initialView: 'dayGridMonth',
        events: 'get_events.php',
        editable: true,
        selectable: true,
        // Add handlers for click, drag, etc.
    });
    calendar.render();
});

Form Fields:

	•	Purpose
	•	Status (pending, confirmed, cancelled, completed)
	•	Notes
	•	Duration (hours)

Submit to PHP:

Form sends data to add_appointment.php and inserts into appointments table.

Display All Appointments:

Use Bootstrap table or card UI.

MySQL Join Example:

SELECT a.*, v.name FROM appointments a
JOIN visitors v ON a.visitor_id = v.visitor_id
ORDER BY appointment_date DESC;
